# 人物肖像 + 数字厚涂油画风格 + 强烈的伦勃朗光影 + 复古温暖的色调。

postive:

杰作，最高画质，超精细细节。一幅充满艺术感的特写肖像，主角是一个20岁左右的美丽东亚女孩，她有柔顺的黑色短发和稀疏的刘海，一双清澈的大眼睛平静地凝视着镜头。她穿着一件简约的白色Polo领衬衫。

画面是数字厚涂油画风格，笔触感强烈且可见。光线采用戏剧性的伦勃朗光，一束柔和的光从侧前方照亮了她的脸庞，在脸颊和鼻梁上形成高光，另一侧则陷入柔和的阴影中，光影对比强烈而富有层次感。

整体色调是温暖的复古棕褐色，饱和度较低，营造出一种安静、内敛、怀旧且略带一丝忧郁的氛围。背景是深色的、模糊的室内环境，以突出主体人物。

navative:

低质量，模糊，变形，丑陋，多余的手指，解剖结构错误，水印，签名，文字
