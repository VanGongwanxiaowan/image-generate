# 通义万相-图像编辑提示词指南：完整攻略

我一直在尝试 Qwen-Image-Edit，说实话… 搞砸和完美编辑的区别就在于提示词。

大多数教程只展示了 2-3 个例子，所以我做了一个完整的提示词手册，你可以直接复制到你的工作流程中。

这涵盖了所有内容：文本替换、对象调整、风格转换、场景交换、角色身份控制、海报设计等等。 如果你一直在为扭曲的脸、丑陋的字体或破坏整个图片的编辑而苦恼，这份指南就能解决这个问题。

⸻

📚 提示词类别

⸻

📝 1. 文本编辑（标牌、标签、海报）

使用这些来替换或更正文本，而不会破坏风格。

```
替换标牌上的文本：
```

“将标牌文本替换为‘盛大开幕’。保留原始字体、大小、颜色和透视。不要更改背景或标牌。”


```
修复包装上的拼写错误：
```

“将蓝色标签的拼写更正为‘氮气’。保留字体系列、颜色和对齐方式。”
```
添加海报标题：
```

“在顶部添加标题‘2025 未来博览会’。将字体样式和颜色与现有设计相匹配。不要与主题重叠。”

⸻

🎯 2. 局部外观编辑

对对象或服装进行小的、精确的更改。

移除不需要的物品：

“从桌子上移除咖啡杯。保持阴影、反射和桌子纹理一致。”

更改服装风格：
“将夹克变成红色皮革。保留褶皱、缝线和光照。”

•	交换颜色/纹理：
“将汽车变成亮黑色而不是银色。保留反射和背景。”

⸻

🌍 3. 全局风格或语义编辑

改变整个外观，但保持结构不变。

•	旋转或重新调整角度：
“旋转雕像以显示后部 180° 视图。保留缺失的胳膊和石头纹理。”

•	风格转换：
“以吉卜力工作室的艺术风格重新渲染这个场景。保留角色身份、服装和布局。”

•	照片级真实感升级：
“将这个铅笔素描场景渲染成照片级真实感的照片。保持姿势、透视和比例不变。”

⸻

🔎 4. 微观/区域编辑

精确地针对微小细节。

•	修复字符笔画：
“在红色框内，将字符‘稽’的下部替换为‘旨’。匹配笔画粗细和书法风格。其他一切保持不变。”

•	小对象替换：
“将孩子手中的苹果换成梨，保持手势和阴影不变。”

⸻

🧍 5. 身份和角色控制

在不破坏特征的情况下保留或交换身份。

•	交换主体：
“将主体替换为戴着太阳镜的男人，保持姿势、服装颜色和背景不变。”

•	在新场景中保留身份：
“将同一角色放置在沙漠环境中。保持发型、服装和面部特征相同。”

•	轻微的面部调整：
“给主体添加眼镜。保持面部、光照和发型不变。”

⸻

🎨 6. 海报和合成设计

用于结构化布局和图形设计编辑。

•	添加标语，不破坏设计：
“在标志下方添加标语‘在 Qwen 中舒适创作’。将排版、间距和风格与设计相匹配。”

•	将草图模型转化为最终海报：
“将这个草图海报布局细化成干净的最终设计。保留布局、文本框和标志位置。”

⸻

📷 7. 相机和灯光控制

像摄影师一样指导 Qwen。

•	更改光照：
“用来自右侧的暖色主光和来自背面的冷色边缘光重新照亮场景。保持姿势和背景不变。”

•	模拟镜头选择：
“使用 35 毫米镜头渲染，景深浅，聚焦于主体的脸部。保留环境模糊。”

⸻

💡 获得出色结果的专业提示

•	始终添加“保持其他所有内容不变” → 避免偏差。

•	使用“保留面部/服装特征”锁定身份。

•	对于文本 → “保留字体、大小和对齐方式”。

•	不要过度编辑。改为链接 2-3 个较小的编辑。

•	使用负面提示 → “无失真，无扭曲文本，无重复面部。”
⸻
