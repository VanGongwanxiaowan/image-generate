# 《How to write AI image prompts like a pro [Oct 2025]》网页总结
## 一、核心主题与适用范围
本文聚焦AI图像提示词（Prompt）撰写技巧，覆盖从基础入门到高级进阶的全流程，适配Midjourney、Stable Diffusion 3.5、ChatGPT（GPT-5/4o）、Flux、Seedream 4.0、LetsEnhance等主流AI图像生成工具，同时包含图像分辨率提升、伦理规范及常见问题解答，帮助用户高效生成高质量AI艺术作品。


## 二、关键提示词撰写原则与技巧
### 1. 基础入门：快速上手步骤
- 用一句话明确目标、主体与使用场景（如“咖啡品牌落地页主视觉图”）。
- 补充4-6个高价值细节，涵盖媒介、风格、光线、构图、氛围、色调（均以短句呈现）。
- 需保证相似度或品牌一致性时，附加参考图并说明保留/修改部分。
- 生成前确定宽高比（如正方形、16:9、4:5），迭代时每次仅修改1个元素（颜色、拍摄距离等）。
- 用“负面提示词”排除冗余元素（如“无水印、无多余手指、无文字”），生成后借助AI图像放大工具提升分辨率。

### 2. 核心结构：三大基础要素
AI图像生成工具对结构化提示词响应更佳，需包含以下三部分，示例：“洛杉矶车流中被困的蝙蝠车，印象派画作，广角镜头”
| 要素 | 说明 | 注意事项 |
|------|------|----------|
| 主体（Subject） | 图像核心焦点，以具体名词为主（如人、杯子、狗） | 避免“爱、正义”等抽象概念，否则生成结果偏差大 |
| 描述（Description） | 补充主体动作、场景、背景等细节，回答“发生了什么、主体在做什么、周围环境如何” | 不可忽略背景描述，如“浣熊读书”可优化为“专业照片：浣熊在图书馆读一本书，近景镜头” |
| 风格（Aesthetic/Style） | 定义视觉风格，含媒介（照片、油画、3D渲染）、艺术流派（印象派、哥特风）、艺术家影响、构图（特写、中景） | 明确风格关键词可大幅提升视觉精准度，如“手持电脑设备”优化为“手持电脑设备，蒸汽波美学，产品摄影” |

### 3. 进阶优化策略
- **语言风格**：用自然、具象的对话式语言，而非关键词列表。例如“狐狸、森林、秋天”优化为“一只好奇的红狐狸在黎明时分探索薄雾笼罩的秋日森林，金色阳光透过彩色树叶，在林间地面投下斑驳阴影，狐狸毛发因晨露略显湿润，冷空气中能看见它的呼吸”。
- **长度与结构实验**：根据工具特性调整提示词长度，短提示（10-20词，如“赛博朋克城市景观，霓虹灯，飞行汽车”）、中提示（30-50词）、长提示（50+词）分别适配不同场景，同时可拆分“主体、环境、光线、颜色、氛围、构图”等高级元素细化描述。
- **权重控制**：部分工具支持关键词加权，Midjourney用“::”（如“蓝天::2”提升蓝天权重），Stable Diffusion用“()”（如“(蓝天:1.5)”）。
- **正向引导优先**：多数工具对正向描述响应更好，仅在必要时用负面提示词（如Stable Diffusion排除低质元素）。
- **参考图与风格修饰**：上传参考图或添加风格修饰词（如“梵高风格”“水彩画”“低多边形”），格式可参考“生成与[艺术家名]风格一致/与[参考图链接]相似的图像”。

## 四、图像质量提升与工具
### 1. 分辨率放大方案
- **生成后放大**：先用AI工具生成基础图像，再用LetsEnhance（最高16倍放大至512兆像素）、Topaz Gigapixel AI（细节保留好）、waifu2x（适配动漫）、Cupscale（支持多模型对比）等工具提升分辨率。
- **工具特性适配**：Midjourney默认1024×1024像素，可内部放大至2048像素；LetsEnhance生成1024×1024像素后，可无质量损失放大至4096×4096像素，甚至更高。

### 2. 后期处理
用Adobe Photoshop、GIMP进行手动修图，或用Remini（人脸优化）、Topaz DeNoise AI（降噪）等AI工具优化细节，关键区域可手动修饰弥补AI不足。

## 六、常见问题（FAQ）
1. **文本提示与图像参考提示的区别**：文本提示用文字描述，不同工具生成结果差异大；图像提示上传参考图，部分场景（如扩展图像）效果更优，部分工具支持两者结合。
2. **生成结果不符预期怎么办**：逐步优化提示词，先测试主体，再逐一添加描述词，若效果差则替换同义描述词，直至接近目标。
3. **好的提示词具备哪些特征**：具体、具象，包含主体、环境、光线、颜色、氛围等细节，用自然语言构建清晰画面。
4. **提示词需多具体**：足够详细但不复杂，根据工具能力调整，通过实验找到“细节平衡点”。
5. **如何描述艺术风格**：结合已知术语+细节，如“梵高《星夜》风格，旋转笔触，鲜艳色彩”“极简水彩画，柔和半透明淡彩”。


## 七、实用模板
- **通用模板**：主体，媒介，风格，光线，构图，氛围，色调。示例：“咖啡师肖像，胶片照片，柔和轮廓光，50毫米特写，温暖氛围，青橙色调”。
- **Stable Diffusion模板**：主体，媒介，风格，光线，颜色，构图，额外细节。负面提示：[需排除元素]。权重：(关键词:1.2)提升，[关键词]降低。
- **Midjourney V7模板**：媒介+风格，主体细节，光线，构图，氛围。示例：“彩色铅笔插画，明亮的橙色加州罂粟花，柔和日光，近景构图，宁静氛围”。
